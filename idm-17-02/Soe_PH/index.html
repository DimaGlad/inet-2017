Подготовка в экзамену

Самооценка прохождения теста по инновациям, формулировка одного вопроса по теме "Интернет-технологии",встречающихся в тесте (выбор одного, выбор многих, упорядочивание, соответствие)

Обещепринятые схемы (протоколы)URL включают:

    ПротоколОписание протокола
    1)ftp a) Потоковый протокол реального времени
    2)http b) Адрес электронной почты
    3)rtmp c) Специальная реализация протокола HTTP
    4)rtsp d) Протокол передачи гипертекста HTTP
    5)https e) Новости Usenet
    6)gopher f) Протокол передачи файлов FTP
    7)mailto g) звонок по указанному телефону
    8)news h) Ссылка на интерактивную сессию Telnet
    i) Протокол Gopher
    j) Имя локального файла
    k) Real Time Messaging Protocol проприетарный протокол

    Для формирования вопроса наиболее известные протоколы, использовались в сети интернет.

    Правильный ответ= 1)f, 2)d, 3)k, 4)a, 5)c, 6)i, 7)b, 8)e.
