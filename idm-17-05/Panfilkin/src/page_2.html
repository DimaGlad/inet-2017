В ходе выполнения задания лабораторной работы было разработано приложение представляющее собой список группы.
Приложение должно отвечать следующим требованиям:
  1. Должен присутствовать список студентов
  2. Возможно фильтрации по любому полю
  3. Переход на личную страницу студента по нажатию на него в списке
  4. Должны быть поля ФИО, Модули, Роли, Команда
  5. Редактирование файла должно осуществляться через json файла, являющимся БД для приложения
  6. В приложении лолжен быть поиск по основным полям
В ходе выполнения работы были задействованы технологии:
  1. HTML
  2. JS
  3. CSS/SCSS
  4. jekyllrb
  5. Materialize styleguides
