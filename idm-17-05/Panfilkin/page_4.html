Вопросы по веб программированию

Вопрос 1. Укажите правильный вариант ответа:
Какая аббревиатура означает понятие "Минимально жизнеспособный продукт"
  1. MVVM
  2. MVP
  3. MVC
  4. MWP
  5. CVP
Вопрос очень простой. MVP (анг. minimum viable product) минимально жизнеспособный продукт. Информация взята из лекций.


Вопрос 2. Укажите правильный вариант ответа:
При помощи какой git команды можно получить именения сделанные в определенном коммите ?
  1. pull
  2. cherry-pick
  3. push
  4. checkout
  5. commit
Данный вопрос относится к классу сложных. Ответ на него можно найте в справочнике гит. Команда git cherry-pick позволяет создать копию определенного комита в текущую ветку по своему хешу, например git cherry-pick 3fe32fgd67jgfy.


Установите последовательность:
Вопрос 3. Установите последовательность действий, необходимых для того чтобы ваши локальные изменения появились в удаленном гит репозитории
  1. git add .
  2. git commit -m "Название комита"
  3. git pull
  4. git push
Стандартный расклад при пуше следующий: первым делом нужно добавить сделанные изменения в коммит командой "git add .". Затем нужно дать названия комиту командой "git commit -m <название>". Затем получить изменения с удаленного репозитори для того чтобы смержить их с нашей локальной версией командой "git pull". Посделнее действие - залить на удаленный репозиторий командой "git push".
